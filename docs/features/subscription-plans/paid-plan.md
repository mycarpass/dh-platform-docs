# Plano Paid

## Visão Geral

O plano **Paid** oferece acesso completo a todas as funcionalidades da plataforma.

- 💳 **Custo:** A definir por tier (Starter, Professional, Enterprise)
- ⏱️ **Duração:** Mensal ou Anual
- 🔄 **Renovação:** Automática a cada ciclo

## Ciclos de Faturamento

### Mensal
- **Período:** 30 dias
- **Renovação:** Automática
- **Cancelamento:** Válido até final do período

### Anual
- **Período:** 365 dias
- **Desconto:** 20% (exemplo)
- **Renovação:** Automática
- **Cancelamento:** Válido até final do período

## Gerenciamento da Assinatura

### Atualizar Plano

```
Configurações → Assinatura → Mudar plano
├─ Downgrade: Próximo período
├─ Upgrade: Imediato (prorrateado)
└─ Mudança de ciclo: Próximo período
```

### Cancelar Assinatura

```
Configurações → Assinatura → Cancelar
├─ Acesso mantido até: Final do período
├─ Dados mantidos: Sim
└─ Reativar: Sim, a qualquer momento
```

### Métodos de Pagamento

```
✅ Cartão de crédito
✅ Boleto
✅ Pix
✅ Fatura (para Enterprise)
```

## FAQ

**P: Posso fazer trial do Paid?**
A: Inicialmente não, mas pode ser considerado.

**P: E se eu cancelar, perco meus dados?**
A: Não, todos os dados são mantidos. Você volta para Free com as limitações.

**P: Qual é a política de reembolso?**
A: A definir com Financeiro.

**P: Posso mudar de ciclo (mensal para anual)?**
A: Sim, na próxima renovação.
